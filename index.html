Boas-vindas ao Cachorros Fofinhos

Quero ver fofura!

O que é este site?

Este projeto foi inspirado em um layout didático de acessibilidade na web. A proposta é demonstrar uma estrutura enxuta com navegação por âncoras, controles de tamanho de fonte, modo de alto contraste e uso correto de semântica HTML para leitores de tela. Assim, pessoas com necessidades diversas conseguem navegar melhor e com mais conforto.

O tema escolhido — cachorros fofinhos — nasce do carinho e do amor por animais de estimação. Aqui celebramos a amizade, a lealdade e a alegria que os cães trazem ao nosso dia a dia, com fotos cuidadosamente selecionadas de repositórios de uso livre.

Você pode reutilizar este código como base para outros temas, mantendo os cuidados de acessibilidade. Os créditos das imagens estão listados no rodapé e no arquivo de créditos.
